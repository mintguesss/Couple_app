# CoupleApp - 專屬情侶互動App

## 專案介紹

CoupleApp 是一個專為伴侶和極少數情侶朋友設計的私人手機應用程式，旨在增進伴侶間的親密感和日常生活互動。這個app完全免費，強調隱私保護，無廣告、無數據販售，僅限已配對用戶使用。

### 主要特色
- **私人專屬**：僅限邀請碼加入，確保數據安全
- **輕量設計**：適合小規模用戶群（<100人）
- **跨平台**：支援iOS和Android
- **有趣互動**：通過遊戲化功能提升日常生活樂趣

### 核心功能
1. **用戶註冊與配對**：簡單註冊，生成邀請碼供伴侶加入
2. **即時訊息**：文字聊天、表情符號、照片分享
3. **共享日曆與提醒**：共同日程管理，自定義推送通知
4. **情侶遊戲**：隨機問題、猜拳等互動遊戲
5. **照片牆與回憶**：共享相冊，隨機回憶功能
6. **心情日記**：每日記錄，伴侶互動
7. **任務管理**：共享待辦清單
8. **健康追蹤**：記錄喝水、飲食、步數等，以植物成長遊戲呈現
9. **緊急聯絡**：一鍵SOS訊息

### 技術架構
- **前端**：React Native + Expo
- **後端**：Firebase Functions (Node.js)
- **資料庫**：Firebase Firestore
- **雲端服務**：Firebase (Auth, Storage, Notifications)
